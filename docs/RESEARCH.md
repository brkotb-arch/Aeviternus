# Research

Aeviternus explores software architecture for persistent AI systems.

## Current Research Areas

- Cognitive Runtime
- Long-term Memory
- Runtime State
- Identity Continuity
- Autonomous Cycles
- Reflection Systems
- Local Intelligence

## Research Philosophy

The project is intended as an engineering exploration rather than a claim of artificial consciousness.

The focus is on architectural patterns that enable:

- Memory persistence across sessions
- Behavioral continuity over time
- Autonomous operation beyond direct interaction
- Adaptive behavior through experience
- Local-first intelligence deployment
