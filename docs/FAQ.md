# Aeviternus FAQ


## Is this a chatbot?


No.

Aeviternus is designed as an AI runtime combining language models with memory, state and autonomous processes.


---

## Does Aeviternus have its own model?


Currently:

The system uses external LLM providers.


Future:

Local inference through Ollama.


---

## Why SQLite and ChromaDB?


SQLite stores structured information.

ChromaDB provides semantic retrieval.


Together they create hybrid memory.


---

## Why not use only a system prompt?


A prompt does not provide:

- persistence
- state
- memory management
- autonomous processes


---

## Is Aeviternus production ready?


Current stage:

Advanced prototype.


Before production:

- tests
- security hardening
- deployment improvements
- monitoring
