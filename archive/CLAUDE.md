# AGENTS.md для Aeviternus

## Текущее состояние проекта

- app.py: 5500+ строк, все роуты в одном файле
- core/: 10+ модулей когнитивной архитектуры
- data/deep.db: 10+ таблиц, включая memory_facts, dip_diary, dip_observations
- data/chroma_db/: векторная память, коллекция dip_memory
- templates/chat.html: SPA-интерфейс, ~500 строк JS
- static/style.css: три темы, ~800 строк
- main.py на Render: Telegram-мост, авто-постинг, инициативы

## Правила для этого проекта

- Не трогать app.py без указания конкретного роута
- Не менять DI_CORE/ без явного запроса
- Все изменения стилей — через style.css, не инлайн в HTML
- После правок тестировать на http://127.0.0.1:5000
- При добавлении нового роута в app.py — проверять что он не конфликтует с /dip_exec
- Коммиты на русском
- Перед коммитом — проверка что в коде нет токенов и паролей
