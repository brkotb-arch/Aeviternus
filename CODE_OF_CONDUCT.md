# Code of Conduct

## Our Standards

The Aeviternus project values:

- respectful technical discussion
- constructive criticism
- engineering excellence
- open research

## Expected Behavior

Contributors should:

- communicate respectfully
- explain technical decisions
- provide useful feedback
- respect project goals

## Unacceptable Behavior

Examples:

- harassment
- personal attacks
- malicious contributions
- intentional disruption

## Enforcement

Maintainers may remove contributions that violate these standards.
